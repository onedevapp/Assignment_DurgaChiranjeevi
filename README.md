# Assignment_DurgaChiranjeevi


Eat food, avoid edges, avoid eating own body.


# How to play
Controls:

To Move Up - Up Arrow

To Move Left - Left Arrow

To Move Down - Down Arrow

To Move Right - Right Arrow
